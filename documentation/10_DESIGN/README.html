📄📐 Design Principles

Core Principles

  1. Clarity First
    - Clear hierarchy
    - Readable typography
    - Intuitive navigation

  2. African Heritage
    - Subtle cultural patterns
    - Warm color palette
    - Community-focused imagery

  3. Professional Trust
    - Clean, modern design
    - Consistent spacing
    - Polished interactions

  4. Accessibility
    - WCAG 2.1 AA compliance
    - High contrast ratios
    - Keyboard navigation
    - Screen reader support

📄📋 Logo Usage Guidelines

Clear Space

Minimum clear space: 1x logo height on all sides ← 1x → ┌─────────┐ 1x│ LOGO │1x └─────────┘ ← 1x →

Minimum Size

Usage Minimum Size
Digital 24px height
Print 0.5 inch height
Favicons 16x16px (icon only)

Don'ts

❌ Don't stretch or distort
❌ Don't change colors (except approved variations)
❌ Don't add effects (shadows, gradients)
❌ Don't place on busy backgrounds
❌ Don't rotate or flip
