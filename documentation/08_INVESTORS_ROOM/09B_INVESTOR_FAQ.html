📄1. GENERAL QUESTIONS

Q1: What is Ujamaa DeFi Platform?

A: Ujamaa is an institutional-grade platform for tokenizing African real-world assets (RWAs). We create diversified liquidity pools that finance African SMEs, and investors earn 10-15% APR yield through yield-bearing ERC-3643 tokens (uLP Tokens).

Key Features:
- Mauritius FSC CIS Manager License (regulated)
- Fireblocks institutional custody
- BIIC (Banque Internationale pour l'Industrie et le Commerce)/MCB bank escrow accounts
- ERC-3643 compliance (KYC/AML enforced)
- 10-15% APR yield (EUROD-denominated)

Q2: Who can invest?

A: Ujamaa is open to:

✓ Institutional Investors: Pension funds, insurance companies, DFIs, family offices
✓ Accredited Individuals: HNWIs meeting €100K minimum investment
✓ Crypto Funds: Institutional crypto hedge funds, VC funds
✓ Diaspora Investors: African diaspora with accredited status

Not Available To:
✗ Retail investors (below €100K)
✗ Investors from sanctioned jurisdictions (OFAC/UN/EU/FATF)
✗ U.S. persons (unless accredited and permitted)

Q3: What is the minimum investment?

Q11: What happens if an SME defaults?

A:
1. Credit Provision: 5% reserve covers expected losses
2. GDIZ (Benin) Guarantee: Primary recourse for repayment
3. Collateral Liquidation: Guarantee Token merchandise sold
4. Legal Enforcement: OHADA law enforcement
5. Investor Impact: Diversification limits single-default impact to ~5% of NAV

Stress Test Results:

- Adverse Case (15% defaults): -12% NAV (reserve drawdown)
- Severe Case (25% defaults): -25% NAV (significant but not catastrophic)

Risk Mitigation:

- Credit provision (5% reserve)
- GDIZ (Benin) guarantee
- Due diligence (8-12 week process)

📄3. TECHNOLOGY & SECURITY

Q12.5: What is the difference between a Tokenizer and a Token?

A: This is a critical distinction that often causes confusion:

Component What It Is Example Who Interacts
Tokenizer Smart contract factory that mints tokens UjamaaAssetTokenizer.sol Platform administrators
Token Digital asset you hold as an investor Ujamaa Asset Token (UAT-XXX) You (the investor)

Simple Analogy:
- Tokenizer = Printing press (factory that produces books)
- Token = The actual book you read and own

As an Investor:
- You never interact directly with the Tokenizer
- You only hold Tokens (the digital asset)
- The Tokenizer is simply the technical mechanism that creates tokens

Example Flow:

UjamaaAssetTokenizer.sol (Smart Contract Factory) ↓ mints Ujamaa Asset Token - GDIZ (Benin) Invoice #001 (Digital Asset) ↓ held by Investor Wallet (You own this)

Q14: What is ERC-3643?

A: ERC-3643 (T-REX Protocol) is a token standard for permissioned securities with:
- On-chain Identity: ONCHAINID verification
- Transfer Restrictions: Only verified wallets can hold
- Compliance Enforcement: Jurisdiction, accreditation, lock-up rules
- Regulatory Audit Trail: All transfers logged

vs. Standard ERC-20:
| Feature | ERC-20 | ERC-3643 |
|---------|--------|----------|
| Permissionless | Yes | No |
| KYC/AML | None | Enforced |
| Transfer Restrictions | None | Configurable |
| Regulatory Compliance | No | Yes |
| Use Case | Utility tokens | Security tokens |

4. Insurance: Claim filed (if covered)
5. Communication: Investors notified within 24 hours

Prevention:

- Bug bounty program
- Real-time monitoring (Forta)
- Timelock on upgrades (7 days)
