📊 Pool KPI Decision Matrix

KPI thresholds and decision framework for pool management

📖 Document Info

Status: ✅ Complete | Format: CSV Data

📑 Overview

This document contains the KPI decision matrix used for pool management decisions. The matrix defines thresholds and corresponding actions for various KPI metrics.

📊 KPI Thresholds

KPIGreenAmberRedAction
NAV Change (Daily)< 2%2-5%> 5%Review if amber, investigate if red
Default Rate< 1%1-2%> 2%Enhanced monitoring if amber, action plan if red
Pool Utilization80-95%70-80% or 95-100%< 70% or 100%Adjust deployment strategy
Concentration< 10%10-15%> 15%Reduce exposure if amber, immediate rebalance if red
Liquidity Ratio> 10%5-10%< 5%Prepare liquidity if amber, emergency measures if red

🔄 Decision Framework

Decision Matrix Logic: ──────────────────── IF KPI = Green: → Continue normal operations → Standard reporting IF KPI = Amber: → Increase monitoring frequency → Notify management → Prepare contingency plan → Weekly reporting IF KPI = Red: → Immediate management notification → Implement action plan → Daily monitoring → Board notification if persists > 5 days

📈 Reporting Frequency

  • Green Status: Monthly reporting
  • Amber Status: Weekly reporting
  • Red Status: Daily reporting until resolved
