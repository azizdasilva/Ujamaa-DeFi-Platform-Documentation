🏠 Remote Work Policy

Policy for remote work and flexible work arrangements

📖 Document Info

Status: ✅ Complete | Classification: Internal

📑 Overview

This policy defines guidelines for remote work and flexible work arrangements at UJAMAA DeFi Platform.

🏠 Remote Work Eligibility

Role TypeRemote EligibilityRequirements
DevelopmentFully RemoteReliable internet, secure setup
OperationsHybridOn-call availability
ManagementHybridTeam availability
ComplianceHybridSecure environment

📋 Remote Work Requirements

Technical Requirements

  • Reliable high-speed internet
  • Dedicated workspace
  • Company-provided equipment
  • VPN connection for sensitive access
  • Backup power supply

Security Requirements

  • Secure network (WPA3)
  • Screen privacy
  • Physical security
  • No public WiFi for work
  • Multi-factor authentication

⏰ Working Hours

Core Hours
  • Core Hours: 10:00 AM - 4:00 PM (local time)
  • Flexible Hours: Outside core hours
  • Meetings: Scheduled during core hours
  • On-Call: Rotating schedule

📊 Performance Management

  • Regular check-ins with manager
  • Clear deliverables and deadlines
  • Outcome-based performance
  • Regular team meetings

💻 Equipment & Support

  • Company provides laptop and peripherals
  • IT support available remotely
  • Equipment replacement program
  • Home office stipend
