✍️ Contributor Guidelines

Guidelines for contributing to the UJAMAA DeFi Platform project

📖 Document Info

Status: ✅ Complete | Audience: All Contributors

📑 Overview

Thank you for contributing to UJAMAA DeFi Platform! This guide explains how to contribute effectively to the project.

🔧 How to Contribute

1. Find an Issue

  • Browse open issues on GitHub
  • Look for labels matching your skills
  • Comment on issue to claim it

2. Create Branch

git checkout develop git checkout -b feature/your-feature-name

3. Make Changes

  • Follow coding standards
  • Write tests for new features
  • Update documentation

4. Submit Pull Request

  • Push your branch to GitHub
  • Create PR against develop branch
  • Fill out PR template
  • Request review from maintainers

✅ PR Checklist

Before Submitting PR
  • ☐ Code follows style guide
  • ☐ Tests added and passing
  • ☐ Documentation updated
  • ☐ No security issues
  • ☐ Performance considered
  • ☐ Error handling complete
