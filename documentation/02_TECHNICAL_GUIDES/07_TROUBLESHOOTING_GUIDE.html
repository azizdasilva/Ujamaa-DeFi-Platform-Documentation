🔧 Troubleshooting Guide

Common issues and their solutions

📖 Document Info

Status: ✅ Complete | Audience: All Users

📑 Overview

This guide helps diagnose and resolve common issues with the UJAMAA DeFi Platform.

🔐 Wallet Connection Issues

ProblemSolution
MetaMask not connectingRefresh page, ensure MetaMask unlocked, check network
Wrong networkSwitch to Polygon Amoy (Chain ID: 80002)
Transaction pendingCheck Polygonscan, increase gas if stuck
Transaction failedCheck error message, ensure sufficient MATIC for gas

💻 Development Issues

Backend Not Starting

# Check Python version python --version # Should be 3.10+ # Check dependencies pip install -r requirements.txt # Check environment cat .env # Ensure all required vars set # Check logs tail -f backend/logs/app.log

Frontend Build Failing

# Clear cache rm -rf node_modules/.cache rm -rf .next # Reinstall dependencies npm install # Check Node version node --version # Should be 18+

⛓️ Smart Contract Issues

ProblemSolution
Deployment failingCheck gas limit, verify private key, check network
Contract call failingCheck function signature, parameters, permissions
Out of gasIncrease gas limit, optimize contract
Revert errorCheck require conditions, check permissions

📊 Database Issues

Common Database Problems
  • Connection refused: Check database is running, verify connection string
  • Migration failed: Check migration files, rollback and retry
  • Slow queries: Check query execution plan, add indexes
  • Lock timeout: Check long-running transactions, optimize queries

🆘 Getting Help

  • Check Logs: Always check application logs first
  • Search Issues: Search existing GitHub issues
  • Slack: Post in #tech-help channel
  • Documentation: Search this documentation site
