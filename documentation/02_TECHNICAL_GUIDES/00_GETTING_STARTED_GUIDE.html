🚀 Getting Started Guide

Quick start guide for new developers joining the Ujamaa DeFi Platform project

📖 Document Info

Level: Beginner | Reading Time: 15 minutes

📑 Welcome to Ujamaa!

This guide will help you get set up and productive on the Ujamaa DeFi Platform project quickly.

🔧 Prerequisites

SoftwareVersionPurpose
Node.js18.x+Frontend development
Python3.10+Backend development
GitLatestVersion control
MetaMaskLatestWeb3 wallet

📥 Setup Steps

1. Clone Repository

git clone cd "Ujamaa DeFi PlatForm"

2. Install Backend Dependencies

cd backend pip install -r requirements.txt

3. Install Frontend Dependencies

cd ../frontend npm install

4. Configure Environment

cp .env.example .env # Edit .env with your configuration
