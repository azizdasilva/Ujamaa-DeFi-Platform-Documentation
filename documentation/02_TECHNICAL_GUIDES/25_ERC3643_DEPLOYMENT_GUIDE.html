🚀 ERC-3643 Deployment Guide

Complete guide to deploying ERC-3643 infrastructure and MVP contracts

📊 Document Info

Version: 2.0.0 | Last Updated: April 2, 2026 | Network: Polygon Amoy (Chain ID: 80002)

🎯 Overview

This guide covers the deployment of ERC-3643 infrastructure and MVP contracts to Polygon Amoy testnet. The deployment includes:

  • ERC-3643 Infrastructure: IdentityRegistry + Compliance Module
  • MVP Contracts: ULPToken (ERC-3643 compliant), LiquidityPool, and supporting contracts
  • Mock Services: MockEscrow, MockFiatRamp, MockEUROD for testing

🔧 Prerequisites

  1. Foundry Installed: curl -L https://foundry.paradigm.xyz | bash
  2. Node.js 18+: For frontend integration
  3. MetaMask Wallet: With test POL tokens
  4. Git: For cloning repository

🚀 Deployment Steps

Step 1: Clone Repository

git clone <repository-url> cd "Ujamaa DeFi Platform"

Step 2: Configure Environment

cd backend cp .env.example .env # Edit .env with your private key

Step 3: Deploy ERC-3643 Infrastructure

forge script script/DeployERC3643.s.sol:DeployERC3643AndMVP \ --rpc-url https://rpc-amoy.polygon.technology/ \ --broadcast \ --verify

Step 4: Verify Deployment

python backend/verify_all_contracts.py
