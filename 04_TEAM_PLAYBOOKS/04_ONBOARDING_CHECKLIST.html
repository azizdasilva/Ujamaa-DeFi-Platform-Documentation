✅ Onboarding Checklist

New team member onboarding checklist

📖 Document Info

Status: ✅ Complete | Audience: New Hires, Managers

📑 Overview

This checklist ensures new team members have everything they need to succeed at Ujamaa DeFi Platform.

📋 Pre-Arrival

Before Day 1
  • ☐ Offer letter signed
  • ☐ Background check completed
  • ☐ Equipment ordered
  • ☐ Email account created
  • ☐ Slack account created
  • ☐ Welcome email sent
  • ☐ Onboarding buddy assigned

📝 Day 1

First Day
  • ☐ Welcome meeting with manager
  • ☐ Office tour (if applicable)
  • ☐ Equipment setup
  • ☐ Access credentials provided
  • ☐ Team introductions
  • ☐ Lunch with team
  • ☐ Review code of conduct

📚 Week 1

First Week Goals
  • ☐ Complete HR paperwork
  • ☐ Set up development environment
  • ☐ Read development methodology
  • ☐ Read Git workflow guide
  • ☐ Complete security training
  • ☐ Meet with key stakeholders
  • ☐ Set up 30-60-90 day goals

📈 Month 1

First Month Goals
  • ☐ Complete first task/bug fix
  • ☐ Attend team meetings
  • ☐ Complete role-specific training
  • ☐ Review project documentation
  • ☐ 30-day check-in with manager
