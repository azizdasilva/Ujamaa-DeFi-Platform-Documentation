1️⃣1. Overview

This document defines the key signals to monitor for each competitor class, recommended tools and sources for ongoing tracking, and the review cadence for competitive intelligence updates.

2️⃣2. Monitoring Signals by Competitor Class

2.1 Class 1 Direct Competitors (Monthly Review)

Competitors: Centrifuge, Goldfinch, Ondo Finance

Signal Category Specific Signals Priority Alert Threshold
Product Launches New lending pools, RWA products, Africa-specific offerings 🔴 Critical Any announcement
Geographic Expansion Africa market entry, OHADA presence, African partnerships 🔴 Critical Any announcement
TVL Changes Significant increases/decreases in total value locked 🟠 High >10% monthly change
Partnership Announcements African institutions, GDIZ-equivalent, banks, regulators 🔴 Critical Any announcement
Regulatory Approvals Mauritius FSC, African licenses, SEC filings 🔴 Critical Any approval
Token Events Major price movements, governance proposals, tokenomics changes 🟠 High >20% price change
Funding Rounds Series A/B/C, strategic investments 🟠 High >$10M raise
Key Hires Africa-focused executives, regulatory experts 🟡 Medium C-level hires
Technology Updates ERC-3643 adoption, Fireblocks integration, Polygon deployment 🟡 Medium Major upgrades
🚨 Class 1 Alert Triggers (Immediate Escalation)
  • Centrifuge announces Africa expansion or African fund
  • Goldfinch announces OHADA-specific lending pool
  • Ondo launches emerging markets credit product
  • Any Class 1 competitor partners with GDIZ, SIPI, or Mauritian entity
  • Any Class 1 competitor obtains Mauritius FSC license

4️⃣4. Review Cadence

4.1 Class 1 Review (Monthly)

Timing: First week of each month
Duration: 2-3 hours
Owner: Competitive Intelligence Lead

Monthly Review Checklist:

  • ☐ Review TVL changes (DeFiLlama)
  • ☐ Check token price movements (CoinMarketCap)
  • ☐ Scan news sources for announcements
  • ☐ Review social media activity (Twitter, LinkedIn)
  • ☐ Check governance forums for proposals
  • ☐ Update competitor profiles if material changes
  • ☐ Flag any alert triggers for immediate escalation
  • ☐ Prepare monthly summary for leadership

4.2 Class 2 Review (Quarterly)

Timing: First week of each quarter (Jan, Apr, Jul, Oct)
Duration: 4-6 hours
Owner: Competitive Intelligence Lead

Quarterly Review Checklist:

  • ☐ Deep dive into each Class 2 competitor (11 total)
  • ☐ Review funding activity (Crunchbase)
  • ☐ Check regulatory filings
  • ☐ Analyze product roadmap changes
  • ☐ Update WSS scores if material changes
  • ☐ Assess convergence risk (Class 2 → Class 1)
  • ☐ Identify partnership opportunities
  • ☐ Prepare quarterly report for leadership

4.3 Class 3 Review (Quarterly)

Timing: Second week of each quarter (following Class 2 review)
Duration: 2-3 hours
Owner: Competitive Intelligence Lead (or delegated analyst)

Quarterly Review Checklist:

  • ☐ Scan news for major developments
  • ☐ Check for Africa-related announcements
  • ☐ Review funding activity (> $50M rounds)
  • ☐ Identify any reclassification candidates
  • ☐ Update profiles if material changes
  • ☐ Light-touch monitoring (no deep dive)

4.4 Competitive Summary Report (Monthly)

Timing: Last week of each month (after Class 1 review)
Duration: 2-3 hours
Owner: Competitive Intelligence Lead

Monthly Summary Report Deliverables:

  • ☐ Executive summary of Class 1 competitor changes
  • ☐ TVL and market movement analysis
  • ☐ Alert trigger assessment
  • ☐ Strategic positioning updates
  • ☐ Recommended actions for leadership
📌 Note

The Competitive Summary Report is classified as a monthly deliverable (marked with 🇲 in navigation). It aligns with Class 1 competitor reviews and provides timely strategic updates for leadership.

4.5 Quarterly Comprehensive Review

Timing: Last week of each quarter (Jan, Apr, Jul, Oct)
Duration: 1-2 days
Owner: Competitive Intelligence Lead + Strategy Team

Quarterly Review Deliverables:

  • ☐ Full Class 1, 2, and 3 competitor assessment
  • ☐ WSS score recalculations
  • ☐ Reclassification recommendations
  • ☐ Market trend analysis
  • ☐ Strategic positioning deep dive
  • ☐ Partnership opportunity assessment
  • ☐ Quarterly report for leadership

4.6 Annual Comprehensive Review

Timing: January each year
Duration: 2-3 days
Owner: Competitive Intelligence Lead + Strategy Team

Annual Review Deliverables:

  1. Full Competitor Landscape Refresh
    • Re-score all competitors using classification framework
    • Identify new competitors not previously tracked
    • Remove competitors no longer relevant
  2. Strategic Positioning Analysis
    • Ujamaa DeFi Platform's competitive positioning vs. each class
    • White space opportunities
    • Threat assessment for coming year
  3. Market Trend Report
    • RWA tokenization trends
    • African DeFi ecosystem evolution
    • Regulatory developments
  4. Updated Monitoring Plan
    • Refresh signal priorities
    • Update tool recommendations
    • Adjust review cadence if needed

5️⃣5. Alert Escalation Protocol

5.1 Alert Levels

Level Description Response Time Escalation Path
🔴 Critical Direct competitive threat Immediate (within 24 hours) CI Lead → CEO → Strategy Team
🟠 High Significant development 48 hours CI Lead → CEO
🟡 Medium Notable but not urgent 1 week CI Lead → Monthly report
🟢 Low Background intelligence Quarterly review CI Analyst → CI Lead

5.2 Critical Alert Examples

Scenario Alert Level Action
Centrifuge announces African fund 🔴 Critical Immediate strategy session
Goldfinch partners with GDIZ 🔴 Critical Immediate partnership counter-strategy
Competitor obtains Mauritius FSC license 🔴 Critical Regulatory response planning
Class 2 competitor raises >$100M 🟠 High Assess competitive response
New ERC-3643 competitor emerges 🟠 High Technology positioning review

5.3 Alert Communication

Critical Alerts:

  1. CI Lead drafts alert memo (1 page max)
  2. Distribute to CEO, Strategy Team via email + Slack
  3. Schedule emergency strategy session (within 48 hours)
  4. Document response plan

High Alerts:

  1. CI Lead posts to #strategy Slack channel
  2. Include in weekly leadership briefing
  3. Assess need for strategy adjustment
