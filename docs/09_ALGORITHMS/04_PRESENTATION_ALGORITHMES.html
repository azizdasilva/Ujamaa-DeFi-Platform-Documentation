Partie 3 : Calcul des Rendements

Modèle de Croissance de la Valeur

Nous utilisons un modèle "Value-Accrual" :

  • Le nombre de tokens UPT reste constant.
  • La valeur de chaque token (NAV) augmente avec les intérêts perçus.
NAV = Valeur Totale du Pool / Nombre de Parts

Partie 4 : Tokenisation des Garanties (UGT)

De l'actif physique au Token

L'Industrial Gateway certifie les stocks (coton, noix de cajou, etc.) avant de créer un Token de Garantie (UGT) non-transférable.

Cycle de vie :
Certification ➔ Minting UGT ➔ Dépôt en Collatéral ➔ Remboursement ➔ Libération
