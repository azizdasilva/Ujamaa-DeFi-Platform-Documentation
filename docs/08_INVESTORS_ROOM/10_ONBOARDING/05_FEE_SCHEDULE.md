# 3. Blockchain/Gas Fees

## 3.1 Network Fees

UJAMAA DEFI PLATFORM operates on **Polygon** for most transactions, offering significantly lower fees than Ethereum mainnet.

| Transaction | Polygon (Estimated) | Ethereum (Estimated) |
|-------------|--------------------|---------------------|
| **Token Transfer** | $0.001–$0.01 | $5–$50 |
| **Claim Distribution** | $0.001–$0.01 | $10–$100 |
| **Wallet Connection** | FREE (off-chain) | N/A |
| **KYC Claim Mint** | $0.01–$0.05 | $20–$100 |
| **Token Approval** | $0.001–$0.01 | $5–$30 |
| **Staking/Unstaking** | $0.01–$0.05 | $10–$50 |

**Note:** Gas fees are estimates and vary based on network congestion. Fees are paid in MATIC (Polygon's native token).

## 3.2 Gas Fee Responsibility

| Transaction | Paid By |
|-------------|---------|
| **Subscription** | Investor (included in total) |
| **Token Transfer** | Sender |
| **Distribution Claim** | Platform (subsidized) |
| **KYC Verification** | Platform (subsidized) |
| **Redemption** | Investor |

## 3.3 Gas Fee Optimization

**Tips to Minimize Gas Costs:**

1. **Batch Transactions:** Combine multiple actions when possible
2. **Off-Peak Hours:** Transact during low network usage (typically weekends)
3. **Polygon Network:** Use Polygon instead of Ethereum when possible
4. **Gas Tokens:** Consider using gas token solutions (if available)
