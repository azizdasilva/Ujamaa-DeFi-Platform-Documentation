## 🎯 Key Concepts

### Tokenization

**Definition:** Converting real-world assets into digital tokens on a blockchain.

**Benefits:**
- ✅ Fractional ownership
- ✅ Increased liquidity
- ✅ 24/7 trading
- ✅ Lower transaction costs
- ✅ Transparent ownership
- ✅ Automated compliance

---

### ERC-3643 (T-REX Protocol)

**Definition:** Permissioned token standard for compliant securities on blockchain.

**Key Features:**
- ✅ Identity verification required
- ✅ Transfer restrictions enforced
- ✅ Regulatory compliance built-in
- ✅ Compatible with ERC-20
- ✅ On-chain compliance

---

### uLP Token (Ujamaa Liquidity Pool Token)

**Definition:** Yield-bearing token representing ownership in a liquidity pool.

**Characteristics:**
- ✅ ERC-3643 compliant
- ✅ Accrues yield automatically
- ✅ Redeemable for underlying assets
- ✅ Permissioned transfers
- ✅ Monthly valuation

## 📖 Glossary

| Term | Definition |
|------|------------|
| **Blockchain** | Distributed ledger technology |
| **Smart Contract** | Self-executing code on blockchain |
| **Tokenization** | Converting assets to digital tokens |
| **ERC-3643** | Permissioned token standard |
| **ONCHAINID** | On-chain identity verification |
| **uLP Token** | Ujamaa Liquidity Pool token |
| **UGT** | Ujamaa Guarantee Token |
| **NAV** | Net Asset Value |
