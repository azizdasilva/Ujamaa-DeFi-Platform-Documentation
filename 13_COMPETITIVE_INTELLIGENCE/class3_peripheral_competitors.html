Ujamaa DeFi Platform — Competitive Intelligence

ℹ️ Class 3 Overview

Last Updated: April 12, 2026 | Total Class 3 Competitors: 14 | WSS Range: 0–34 | Monitoring Cadence: Quarterly reviews (lighter touch)

📋Overview

Class 3 competitors are distant players in the broader DeFi or African fintech space with limited direct overlap:

  • Global DeFi protocols with minimal Africa presence
  • African fintechs without blockchain/DeFi features
  • Pure crypto exchanges without lending/tokenization
  • Different user segments (retail traders vs. institutional investors)
📅 Monitoring Cadence

Quarterly reviews (lighter touch than Class 1/2)

1. Aave

One-Paragraph Summary: Aave is the largest decentralized lending protocol in DeFi, with $57B+ TVL (January 2026). While Aave dominates global DeFi lending, it has minimal Africa-specific presence and serves a different user segment (crypto-native lenders/borrowers vs. institutional investors in African SMEs).

Attribute Detail
Website https://aave.com
Founded 2020 (Aave V2)
TVL $57B+ (January 2026)
Blockchain Ethereum, Polygon, Avalanche, others
Token AAVE (governance)
Africa Presence Minimal (some South African DeFi integration)

WSS: 40.0 → Class 3 (Peripheral) — Borderline Class 2

Reason for Class 3 Classification:

  • No Africa-focused products or partnerships
  • Crypto-collateralized lending (not RWA)
  • Retail/DeFi-native user base

Reclassification Triggers:

  • Aave launches Africa-specific lending pool
  • Partnership with African institutional entities
  • RWA collateral integration for African assets
