🧮 Worked Example: Scoring Centrifuge

Step-by-step demonstration of the WSS methodology applied to a real competitor

📅 Document Info

Version: 1.0 | Date: March 31, 2026 | Purpose: Demonstrate practical application of the classification framework

🎯 Target Competitor

Centrifuge — Decentralized asset financing protocol enabling real-world assets to be used as collateral in DeFi

TVL: $1.3B+ | Founded: 2017 | HQ: Berlin, Germany

1️⃣1. Evidence Collection

Before scoring, we gather verifiable evidence from multiple sources. Each criterion requires documented proof.

1.1 Evidence Summary Table

Criterion Evidence Found Source Confidence
Geography Global platform; ~15% African exposure through real-world asset pools Centrifuge Q1 2026 Report, docs.centrifuge.io 🟢 High
User Segment Institutional investors + SME borrowers (via asset originators like Altifi, Paperchain) Product documentation, investor deck 🟢 High
Product Liquidity pools (Tinlake), RWA tokenization, yield-bearing tokens (DROP/TIN), compliance via ONCHAINID docs.centrifuge.io, app.tinlake.io 🟢 High
Tech Stack Multi-chain (8 chains including Ethereum, Polygon); EVM-compatible; custom NFT standard Technical documentation, GitHub 🟢 High
Business Model Protocol fees (0.5-2%), performance fees on pools, origination fees Token economics docs, fee schedule 🟡 Medium
Social Mandate "DeFi for the real economy" mission; no explicit UN SDG alignment or measurable targets Website, whitepaper, blog posts 🟢 High
Token Economy CFG governance token; staking for rewards; not yield-bearing directly; no ERC-3643 compliance Token docs, CoinMarketCap 🟢 High
Partnerships Fireblocks, Chainlink, Apollo, Janus Henderson, Consensys Partnership announcements, press releases 🟢 High
📌 Evidence Quality Notes
  • 🟢 High Confidence: Multiple corroborating sources (official docs + third-party reports)
  • 🟡 Medium Confidence: Single source or self-reported data requiring verification
  • 🔴 Low Confidence: Unverified claims; requires follow-up research

2️⃣2. Scoring Rationale

Each criterion is scored 0-5 based on the rubric in the Classification Framework.

2.1 Detailed Scoring

# Criterion Score Weight Justification
1 Geography 3 20% Centrifuge has 15-30% African exposure through asset pools. Not primary focus, but expanding. → Moderate Overlap
2 User Segment 5 15% Identical target: institutional investors (€100K+) + SME borrowers (via originators). → Identical
3 Product Overlap 5 20% 7+ overlapping features: liquidity pools, RWA tokenization, yield-bearing tokens (DROP/TIN), compliance/KYC, institutional custody, bank integration. → Identical
4 Tech Stack 3 10% Same L1/L2 (Ethereum, Polygon) but different token standards (custom NFT vs. ERC-3643). EVM-compatible. → Moderate Overlap
5 Business Model 4 10% Very similar: protocol fees (0.5-2%) + performance fees + origination fees. Slightly different structure. → High Overlap
6 Social Mandate 3 10% Clear mission ("DeFi for the real economy") but no measurable SDG targets or job creation goals. → Moderate Overlap
7 Token Economy 2 8% CFG is governance-only with staking rewards. Not yield-bearing security token. No ERC-3643 compliance layer. → Low Overlap
8 Partnerships 4 7% 5+ shared partner categories: Fireblocks (custody), Chainlink (oracle), institutional players (Apollo, Janus Henderson). → High Overlap

2.2 Score Validation Checklist

✅ Validation Complete
  • ✅ All scores have cited evidence sources
  • ✅ Scores align with rubric descriptors
  • ✅ Confidence levels documented
  • ✅ Ready for peer review (Class 1 scoring requires review)

3️⃣3. WSS Calculation

3.1 Step-by-Step Calculation

🧮 Formula Application
WSS = Σ(Score × Weight)

Geography:        3 × 20% = 0.60
User Segment:     5 × 15% = 0.75
Product Overlap:  5 × 20% = 1.00
Tech Stack:       3 × 10% = 0.30
Business Model:   4 × 10% = 0.40
Social Mandate:   3 × 10% = 0.30
Token Economy:    2 ×  8% = 0.16
Partnerships:     4 ×  7% = 0.28
                          ───────
                    Sum:    3.79

WSS = 3.79 × 20 = 75.8

3.2 Classification Result

✅ Final Result

Centrifuge WSS = 75.8

Classification: Class 1 (Direct Competitor)

Threshold met: 65 ≤ 75.8 ≤ 100

3.3 Sensitivity Analysis

What if we scored more conservatively?

Scenario Adjusted Scores New WSS Classification
Base Case As scored above 75.8 Class 1
Conservative Geography -1, Product -1 67.8 Class 1
Aggressive Tech Stack +1, Partnerships +1 83.8 Class 1

Conclusion: Classification is robust across reasonable scoring variations. Centrifuge remains Class 1 in all scenarios.

4.2 Strategic Implications

  • Ujamaa's Moat: Africa-first focus + ERC-3643 compliance + regulatory licensing
  • Centrifuge's Moat: Scale ($1.3B TVL) + multi-chain deployment + established partnerships
  • Battlefield: African SME financing, institutional investor attention, compliance credibility

  • 🔴 Centrifuge launches OHADA-specific asset pool

  • 🔴 Centrifuge announces Africa-focused fund (>€50M)

5.3 Response Playbook

If red flags trigger:

  1. Immediate: Alert leadership team within 24 hours
  2. Week 1: Deep-dive analysis on threat specifics
  3. Week 2: Strategic response recommendations
  4. Week 3: Execute countermeasures (partnership announcements, PR, etc.)
